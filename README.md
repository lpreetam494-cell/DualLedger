## 🚀 Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/lpreetam494-cell/DualLedger.git
   ```
2. Navigate into the project directory:
   ```bash
   cd DualLedger
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠 Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **State Management:** Zustand
- **Icons:** Lucide React
- **Charts:** Recharts
- **Animations:** Framer Motion
